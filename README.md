## Foundry Smart Contract Development - Fund Me

This repository contains the "Fund Me" smart contract project developed as part of the Cyfrin Updraft Foundry module. It demonstrates modern smart contract development and testing practices using the Foundry framework. The project focuses on integrating Chainlink Price Feeds for real-world data, writing robust unit tests with Forge, and automating deployments via Solidity scripting. It highlights practical experience with dependency management, local Anvil network interaction, and foundational gas optimization techniques.

## Foundry

**Foundry is a blazing fast, portable and modular toolkit for Ethereum application development written in Rust.**

Foundry consists of:

- **Forge**: Ethereum testing framework (like Truffle, Hardhat and DappTools).
- **Cast**: Swiss army knife for interacting with EVM smart contracts, sending transactions and getting chain data.
- **Anvil**: Local Ethereum node, akin to Ganache, Hardhat Network.
- **Chisel**: Fast, utilitarian, and verbose solidity REPL.

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ forge script script/Counter.s.sol:CounterScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```

### Cast

```shell
$ cast <subcommand>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
